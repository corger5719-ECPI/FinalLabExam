Application Menu Project


Project Overview

After completing this project assignment, I interpreted the application requirements and developed Python code to implement an application menu that serves as a building block for future assignments throughout the course. This project provided practice with user input, menu creation, program flow, and basic Python programming concepts.

Project Purpose

The purpose of this project was to create a menu-driven application that allows a user to interact with different program options. The menu structure established in this assignment can be expanded and reused in future projects as additional functionality is added.

Learning Objectives

This project demonstrates the ability to:

* Create a menu-driven application in Python.
* Display information to the user.
* Accept and process user input.
* Use variables to store user selections.
* Implement decision-making structures.
* Display the current date and time.
* Develop reusable program components for future assignments.

Program Features

The application menu includes the following options:

1. Input Data
2. View Current Data
3. Generate Report

The program prompts the user to select an option and displays the selected menu choice along with the current date and time.

Technologies Used

* Python
* IDLE Development Environment
* datetime module


Skills Demonstrated

During this project, I gained experience with:

* Python syntax
* Variables and data types
* User input
* Output formatting
* Program organization
* Menu design
* Date and time functions
* Basic application development
* Graphing Dynamically Generated Data

Conclusion

This project provided valuable experience in creating a structured Python application. The menu system developed in this assignment establishes a framework that can be expanded throughout the course as more advanced programming concepts are introduced.
